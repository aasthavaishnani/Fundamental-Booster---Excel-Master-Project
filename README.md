# 🚀 PR. 1 Fundamental Booster - Excel Master Project

This project is designed to master core Excel fundamentals and advanced data analysis formulas. It involves performing various complex calculations on real-world datasets, including Student Grades, Sales Data, and Employee Records.

## 📊 Project Overview
The project is divided into three main analytical sheets:

1. **Students Grade Analysis**: Focuses on grading systems, performance tracking, and text manipulation.
2. **Sales Data Management**: Handles financial rounding, tax calculations, and discount eligibility logic.
3. **Employee Record Tracking**: Manages joining date analysis, experience calculation, and dynamic salary lookups.

---

## 🛠️ Key Formulas & Logic Applied

### 1. Advanced Logical Formulas
* **Nested IF**: Classified student grades (A+, A, B, C, Fail) based on their average scores.
* **IF with AND**: Identified "Top Performers" who scored above 80 in both Math and Science.
* **IF with OR**: Determined "Discount Eligibility" in Sales data if the product is a "Laptop" OR the amount exceeds 50,000.

### 2. Lookup & Reference Functions
* **VLOOKUP**: Used to fetch product prices and student names based on specific IDs.
* **XLOOKUP / INDEX-MATCH**: Implemented for dynamic employee salary searches and multi-criteria lookups.
* **Absolute Reference ($)**: Applied to keep the GST Rate (18%) cell fixed during bulk calculations.

### 3. Text & Data Cleaning
* **UPPER/LOWER**: Standardized names into uppercase and lowercase formats.
* **LEFT & FIND**: Extracted the "First Name" dynamically from a full name string.
* **Proper Formatting**: Applied bold headers, currency symbols (₹), and standard date formats.

### 4. Math & Date Functions
* **DATEDIF**: Calculated "Service Years" (Experience) by comparing the Joining Date to the current date (`TODAY`).
* **ROUND, CEILING, FLOOR**: Used for financial precision—rounding amounts to the nearest whole number or specific intervals.
